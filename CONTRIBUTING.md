# Contributing Guidelines

This project was built as an internship assignment. For any improvements or suggestions, please follow these guidelines:

1. **Code Style**: Follow TypeScript and React best practices
2. **Commits**: Write clear, descriptive commit messages
3. **Testing**: Test your changes thoroughly before submitting
4. **Documentation**: Update README.md if you add new features

## Development Workflow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description

Thank you for your interest in improving this project!
